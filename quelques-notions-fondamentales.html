Quelques notions fondamentales

Internet est un réseau mondial qui connecte des ordinateurs et autres appareils et permet la transmission de données. (il comprend le web, les emails, mais pas que…)

Le web (World Wide Web) est un service d'Internet qui permet d'accéder à des pages web

Page Web : Document créé en HTML (structure), CSS (style) et JavaScript (interactions)

URL : l’adresse unique d'une page web

Serveur : Ordinateur qui stocke et envoie les pages web au navigateur

Moteurs de recherche : Aident à trouver des pages web.

HTTP et HTTPS : Protocoles qui permettent d'échanger des données (HTTPS est sécurisé car il crypte les données)

Liens : Éléments cliquables pour naviguer entre les pages. On parle de liens hypertexte.
